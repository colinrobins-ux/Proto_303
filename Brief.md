#The Ivy Mobile contact form

##Basics
This is a mobile-first, responsive website who's goal is to generate lead information for a wedding venue, in this case, The Ivy. There will be a few sections. First, there should be a nice big hero image (images/logo.png) of the building itself, with the text "The Ivy" overlaid in a classy, luxury type design. Below that, there should be a list of services we offer, such as concerts, quincenearas, weddings, and other things you can recommend and add in. Finally, I want a CTA that says inquire. When a user clicks that, they should see a very basic form. The form should have name, event, a possible date, and a comment box for any additional information. When the user submits that form, an email should be generated and sent to rcolinrobins@gmail.com
